Internal 1

Silhouette Coefficient

"Am I closer to my own cluster or to the nearest other cluster?"

How It Works

  1. a(i) = mean distance to same-cluster points. (Cohesion)
  2. b(i) = mean distance to nearest other cluster. (Separation)
  3. s(i) = (b−a)/max(a,b)  ∈ [−1, +1]
Per-sample Silhouette
s(i) = ( b(i) − a(i) ) / max( a(i), b(i) )

External 3

V-Measure

Homogeneity (purity) + Completeness (coverage) → harmonic mean.

HOMOGENEITY h

Clusters pure? h = 1 − H(U|V)/H(U)

COMPLETENESS c

Classes whole? c = 1 − H(V|U)/H(V)

V-Measure
V = 2·h·c / (h+c)

External 4

Fowlkes-Mallows Index

√(Pair-Precision × Pair-Recall)

PAIR PRECISION

P = TP/(TP+FP)

PAIR RECALL

R = TP/(TP+FN)

Fowlkes-Mallows
FM = √(P × R) ∈ [0,1]
💡 Low P → impure (merging). Low R → fragmented (splitting).

Workflow

  1. No labels: CH sweep → Silhouette → DB weak boundaries.
  2. Labels: ARI primary → V-Measure (h,c) diagnostics → AMI if comparing k.
  3. Always test multiple algorithms on same data. Try non-convex datasets.
